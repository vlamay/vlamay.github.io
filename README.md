## Structure

```
.
├── index.html          # redirects to Portfolio.html
├── Portfolio.html      # main portfolio (scroll-driven, dark cinematic)
├── resume.html         # print-ready resume with PDF download
├── css/
│   ├── portfolio.css
│   └── resume.css      # includes @media print rules for A4 PDF
├── js/
│   ├── portfolio.js
│   └── resume.js
└── .nojekyll           # disable Jekyll, serve files as-is
```

## Deploy to GitHub Pages

1. Commit all files to the `main` branch of `vlamay/vlamay.github.io`.
2. In **Settings → Pages**, set source to `main` / `/ (root)`.
3. Site goes live at `https://vlamay.github.io/` within ~60 seconds.

No build step required — plain HTML/CSS/JS, static hosting only.
